# GradesApp
A simple app where you can log and track your grades.
Built with Python (guizero).

## Requirements
- Python 3.x
- guizero
- importlib

## How to Run
1. Install *guizero*: `pip install guizero`
2. Install *importlib*: `pip install importlib`
3. Run the Code: `python gradesApp.py`
